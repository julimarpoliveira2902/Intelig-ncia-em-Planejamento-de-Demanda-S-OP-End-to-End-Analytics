# Intelig-ncia-em-Planejamento-de-Demanda-S-OP-End-to-End-Analytics
Este projeto consiste em uma solução **End-to-End de Engenharia e Análise de Dados** desenvolvida para resolver desafios reais de **Planejamento de Demanda, S&amp;OP (Sales and Operations Planning) e Gestão de Estoques** na indústria de bens de consumo (FMCG).
